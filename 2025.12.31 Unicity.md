#### alphaminer 锄头

> 其他配置参数 (可选)
>
> ```ini
> largepages=true
> threads=40
> ```
>
> - `threads=40`: 指定线程数，如果不指定则使用所有线程
> - `largepages=true`: 启用大页内存（可能提高性能）

```json
{
    "flightName": "Unicity",
    "descMsg": "Unicity",
    "digitalCash": "Unicity",
    "miningPool": "1",
    "miningConfig": "Custom",
    "disableFaultCard": false,
    "customConfig": {
        "customMiner": "alphaminer",
        "customInstallUrl": "http://114.215.172.228:8899/unicitynetwork/alphaminer-v4.0.0.e.tar.gz",
        "customAlgo": "---",
        "customTemplate": "%WAL%.%WORKER_NAME%",
        "customUrl": "stratum+tcp://150.5.155.23:3054",
        "customUserConfig": "largepages=true\nthreads=40"
    }
}
```
