```sh
## 生成新的秘钥对
nockchain-wallet keygen

## 导入私钥
# 从 jam 文件中
nockchain-wallet import-keys --input path/to/keys.jam
# 导入主公钥和链码
nockchain-wallet import-master-pubkey --key <base58-key> --knot <base58-chain-code>

## 从助记词生成私钥
nockchain-wallet gen-master-privkey --seedphrase "your seed phrase here"

## 使用私钥生成公钥
nockchain-wallet gen-master-pubkey --master-privkey <private-key>

## 派生子密钥（Child Key）
## 从主密钥派生子密钥。你可以指定派生公钥或私钥，以及子索引（最多 256 个子密钥）。
nockchain-wallet derive-child --key-type <pub|priv> --index <0-255>
```

##  H9 池

池网站：https://www.h9.com/

#### x-proxy (可选)

- 这里的钱包也要使用 H9 的 API Key
- 只需要一台机器运行机器，其他机器连接到这台
- **注意：需要单独创建一个钱包，币种写 NockxProxy**

```json

        "customTemplate": "%WAL%"
    }
}
```



#### 飞行表

> 其他配置参数：
>
> - `threads=1`: 指定线程数。如果写成 0 代表使用所有线程
> - `nock_proxy`: 指定 xProxy
> - `auto_threads`: 自动计算线程
>   - 如果 (内存 > 线程 * 2) ，则实际线程数使用 CPU 线程数
>   - 如果 (内存 < 线程 * 2) ，则实际线程数使用 (内存 / 2)
>
>
> ```ini
> threads=1
> nock_proxy=http://192.168.1.12:6899
> auto_threads=true
> ```
