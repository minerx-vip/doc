关于时间同步：

- 时间同步：Olivetum 高度依赖精确的时间。如果你的系统时钟偏差超过几秒，你的节点可能会拒绝区块并停止同步
- 因系统时间错误导致的讹典型错误：
  - block in the future
  - timestamp increment below minimum



下载挖矿软件

- 选项 A (推荐)：从官方 Discord 下载预构建的 `geth` 二进制文件和 `olivetum_pow_genesis.json`
- 选项 B (高级)： 从源码编译



准备你的数据目录：

```sh
DATADIR=~/olivetum_pow
mkdir -p "$DATADIR/keystore"
```



初始化链：

```sh
./geth init --datadir "$DATADIR" olivetum_pow_genesis.json
```



开始 Solo 挖矿：

```sh
geth \
 --datadir "$DATADIR" \
 --networkid 30216931 \
 --syncmode full \
 --http --http.addr 127.0.0.1 --http.api eth,net,web3,txpool \
 --ws --ws.addr 127.0.0.1 --ws.api eth,net,web3 \
 --mine \
 --miner.threads 1 \
 --miner.etherbase 0x72eb2f74ba139700510449954d3bc553a5f83038
```



检查状态：

```sh
eth.mining
eth.blockNumber
eth.getBalance("0xYOUR_ADDRESS")
```



高 CPU 使用率

```sh
## 减少线程数
```
