## CLI 钱包使用

```sh
# 一次性设好变量(钱包加载名就是它的完整路径)
CLI=/home/ubuntu/btx/bin/btx-cli
W="/home/ubuntu/btx-wallet/btx_xxx"

# 【仅重启后需要】先加载钱包
$CLI loadwallet "$W"

# 余额总览:balance(可花)/ immature_balance(未成熟)/ shielded_balance / txcount
$CLI -rpcwallet="$W" getwalletinfo

# 透明余额明细:trusted=可花  immature=未成熟  untrusted_pending=未确认  settlement_safe=结算安全(≥12确认)
$CLI -rpcwallet="$W" getbalances

# 透明+屏蔽合计(只统计“可花”,未成熟不计)
$CLI -rpcwallet="$W" z_gettotalbalance

# 只要一个“可花余额”数字
$CLI -rpcwallet="$W" getbalance

## 转账
$CLI -rpcwallet="$W" sendtoaddress btx1zjv3mfdka9wy0hrzu93xfjxxx 90

## 指定 fee
$CLI -rpcwallet="$W" -named sendtoaddress \
    address="btx1zjv3mfdka9wy0hrzu93xfjxxx" \
    amount=0.5 \
    fee_rate=20 \
    replaceable=true

## 提高 GAS
## 当前这笔交易费率约 1.92 atom/vB，可以直接 bump 到比如 10 atom/vB：
$CLI -rpcwallet="$W" -named bumpfee \
    txid="2a5f25949f8eac1aef57edeb73b76649e25796a209c35668f9ce59c52cdfc475" \
    fee_rate=50
```
