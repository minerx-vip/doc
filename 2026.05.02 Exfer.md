## 飞行表

#### 幸运池

矿池：https://exfer.luckypool.io/

> 将其他配置参数重的 `<钱包地址>` 更改为自己的钱包地址
>
> 算力：
>
> - 5090：3800H/s
> - 4090：2600H/s
> - 3070Ti：850H/s

```json
{
    "flightName": "exfer",
    "descMsg": "exfer",
    "miningPool": "1",
    "miningConfig": "Custom",
    "disableFaultCard": true,
    "customConfig": {
        "customMiner": "lpminer",
        "customInstallUrl": "http://flight.minerx.vip/exfer/lpminer-0.0.38.tar.gz",
        "customAlgo": "exfer",
        "customTemplate": "%WAL%.%WORKER_NAME%",
        "customUrl": "stratum+tcp://exfer.luckypool.io:3335",
        "customUserConfig": "--algo exfer --pool stratum+tcp://exfer.luckypool.io:3335 --wallet <钱包地址> --worker %WORKER_NAME%"
    }
}
```

## 常用命令

```sh
# 查看当前节点高度
curl -s http://89.127.232.155:9334 -d '{"jsonrpc":"2.0","id":1,"method":"get_block_height","params":{}}' | jq

# 查看指定钱包余额（需RPC运行）
curl -s http://89.127.232.155:9334 \
  -d '{"jsonrpc":"2.0","id":1,"method":"get_balance","params":{"address":"e31b1d2df17a105ea20940f477XXX"}}' \
  | jq '{address: .result.address, balance: (.result.balance / 100000000)}'

## 查看钱包信息 (钱包地址、公钥)
/exfer wallet info --wallet ~/exfer-wallet.key

## 转账命令
./exfer wallet send \
  --wallet /root/.exfer/exfer-wallet-50.key \
  --to e31b1d2df17a105ea20940f477XXX \
  --amount 2902.5EXFER \
  --fee 0.001EXFER \
  --rpc http://127.0.0.1:9334
```
