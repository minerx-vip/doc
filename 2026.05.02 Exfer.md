## 飞行表

#### 幸运池

矿池：https://exfer.luckypool.io/

> 将其他配置参数重的 `<钱包地址>` 更改为自己的钱包地址

```json
{
    "flightName": "exfer",
    "descMsg": "exfer",
    "miningPool": "1",
    "miningConfig": "Custom",
    "disableFaultCard": true,
    "customConfig": {
        "customMiner": "lpminer",
        "customInstallUrl": "https://storage.luckypool.io/lpminer/lpminer-0.0.10.tar.gz",
        "customAlgo": "exfer",
        "customTemplate": "%WAL%.%WORKER_NAME%",
        "customUrl": "stratum+tcp://exfer.luckypool.io:3335",
        "customUserConfig": "--algo exfer --pool stratum+tcp://exfer.luckypool.io:3335 --wallet <钱包地址> --worker %WORKER_NAME%"
    }
}
```
