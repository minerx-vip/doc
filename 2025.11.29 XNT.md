#### DRPool

矿池：https://drpool.io

配置说明：https://github.com/0xdrpool/xnt_gpu_guesser

**根据自己实际情况调整其他配置参数**

```json
{
    "flightName": "xnt",
    "descMsg": "drpool",
    "digitalCash": "xnt",
    "miningPool": "1",
    "miningConfig": "Custom",
    "disableFaultCard": true,
    "customConfig": {
        "customMiner": "xntprover",
        "customInstallUrl": "http://flight.minerx.vip/xnt/xntprover-1.0.0.a.tar.gz",
        "customAlgo": "",
        "customTemplate": "%WAL%.%WORKER_NAME%",
        "customUrl": "stratum+tcp://xnt.drpool.io:30120",
        "customUserConfig": "-m 1 -g 0,1,2,3,4,5"
    },
    "isDefault": false
}
```
