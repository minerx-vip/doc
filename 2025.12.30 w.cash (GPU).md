## 钱包

```sh
## 安装依赖
apt install -y build-essential clang libclang-dev protobuf-compiler
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

## 下载
wget https://w.cash/downloads/wcash.zip
unzip wcash.zip

## 生成钱包
cargo run -p zcash-keygen --bin zcash-keygen -- --mnemonic
```
