## 常用命令

如果使用 Solo，开始挖矿的时候会自动在节点上创建透明地址

```sh
## 获取助记词
./junocash-cli z_getseedphrase

## 获取透明地址余额
./junocash-cli getbalance

## 获取总余额 （透明 + 私密）
./junocash-cli z_gettotalbalance

## 查看区块高度
./junocash-cli getblockcount

## 查看挖矿地址，可以添加到 node
junocash-cli t_getminingaddress

## 重新索引
junocashd -reindex

## 重新同步 - 比重新索引快
junocashd -resync

## 查询钱包信息，包括不成熟的
junocash-cli getwalletinfo
```



## 转账流程

#### 1. 查询余额

```sh
# 查看透明地址余额（挖矿奖励在这里）
junocash-cli getbalance

# 查看总余额（透明 + 屏蔽）
junocash-cli z_gettotalbalance

## 查询钱包信息，包含不成熟
junocash-cli getwalletinfo

## 获取地址列表
junocash-cli listaddresses

## 列出隐私地址
junocash-cli z_listaccounts
```



#### 2. 获取 Orchard 地址

```sh
# 首次使用需要创建账户
junocash-cli z_getnewaccount

# 获取统一地址（以 j1... 开头）
junocash-cli z_getaddressforaccount 0
```



#### 3. Shield 挖矿奖励

**重要**：挖矿奖励（coinbase）需要 **100 个确认** 才能 shield。

```sh
# 将所有透明地址的 coinbase 转到你的统一地址
junocash-cli z_shieldcoinbase "*" "j1fama5nhrrufhv9xxx"
```

这会返回一个操作 ID，检查状态：

```sh
junocash-cli z_getoperationstatus
```

等待状态变为 `success`。



#### 4. 查询屏蔽后的余额

```sh
# 查看指定地址的余额
junocash-cli z_getbalance "j1你的统一地址..."

# 查看未花费的屏蔽 notes
junocash-cli z_listunspent
```



#### 5. 转账

```sh
# 发送到另一个地址
junocash-cli z_send "fromaddress" "sendaddress" 574

# 例如发送 10 JUNO
junocash-cli z_send "j1FromAddr..." "j1ToAddr..." 10.0
```

检查转账状态：

```sh
## 查看交易状态
junocash-cli z_getoperationstatus

## 查看具体交易状态
./junocash-cli z_getoperationstatus | jq '.[] | select(.id=="opid-e16ba642-4af8-46ae-a352-29da7f84fbdb")'

## 查询某一笔交易详情
junocash-cli z_viewtransaction "txid..."
```



```sh
## 列出钱包最近交易
junocash-cli listtransactions
```



#### 常见问题

- **"Insufficient funds"** → 检查 coinbase 是否有 100+ 确认
- **余额显示 0** → 可能还没到 100 确认，用 `junocash-cli getblockcount` 查看当前高度
